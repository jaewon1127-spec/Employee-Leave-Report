# 휴가대장 생성기

인원DB와 휴가DB 엑셀 파일을 올리면 연도별 **휴가대장 / 기타휴가 리스트** 시트가 담긴 엑셀 파일을 만들어 주는 단일 HTML 도구입니다. 설치가 필요 없고, 모든 처리는 사용자의 브라우저 안에서만 이루어집니다.

## 주요 기능

- **연도별 휴가대장 자동 생성** — 휴가DB의 연도를 자동 감지하여 `휴가대장(2026)`, `휴가대장(2025)` … 시트를 생성하며, 기존 수기 양식과 동일한 서식(색상·테두리·병합·표시형식)으로 출력됩니다.
- **기타휴가 리스트** — 표준 휴가구분에 속하지 않는 기록을 연도별 `기타휴가(YYYY)` 시트로 함께 생성합니다.
- **데이터 수정 화면** — 업로드한 인원DB/휴가DB를 화면에서 직접 수정·추가·삭제할 수 있고, 수정본을 엑셀로 다시 내려받을 수 있습니다.
- **자동 저장** — 업로드·수정 내용이 브라우저(localStorage)에 자동 저장되어, 다음에 열면 그대로 복원됩니다.
- **백업 / 복원** — 전체 데이터를 JSON 파일 하나로 내려받고, 다른 PC에서 복원할 수 있습니다.
- **양식 다운로드** — 인원DB / 휴가DB 입력 양식(작성 안내 포함)을 내려받을 수 있습니다.
- **사용 설명서 내장** — 화면 우측의 "사용 설명서" 탭에서 사용법·집계 규칙·FAQ를 볼 수 있습니다.

## 사용 방법

### 방법 1 — GitHub Pages (권장)

저장소 **Settings → Pages → Branch: `main` / 폴더: `/ (root)`** 로 설정하면 아래 주소로 접속할 수 있습니다.

```
https://<사용자명>.github.io/<저장소명>/
```

### 방법 2 — 파일로 사용

`index.html` 하나만 내려받아 더블클릭으로 열면 됩니다. (인터넷 연결 필요 — 엑셀 처리 라이브러리를 CDN에서 불러옵니다)

## 집계 규칙

기존 휴가대장 엑셀 수식과 1:1 검증을 거쳐 동일하게 동작합니다.

| 휴가구분 | 집계 방식 |
|---|---|
| 연차 · 반차 · 여름휴가 | 연차에서 차감 |
| 월차 | 월차에서 차감 (2017.5.30 이후 입사자, 입사 후 12개월간 최대 11일) |
| 병가 · 예비군 · 민방위 · 경조휴가 · 특별휴가 | 별도 항목으로 집계 |
| 대체휴무(반차/연차) · 무급휴가 · 만료된 연차 | 연도별 내역에만 표시 |
| 그 외 모든 구분 | 기타휴가로 집계 + 기타휴가(연도) 시트에 목록 표시 |

- 연차 발생: 근속 1년당 15일, 2년마다 1일 가산, 최대 25일
- 정산기준일: 지난 연도 = 해당 연도 12월 31일, 올해 = 오늘
- 부여기준일: 각 연도의 1월 1일

## 데이터와 보안

- 업로드한 엑셀과 입력한 데이터는 **서버로 전송되지 않으며**, 사용자의 브라우저 안에서만 처리·저장됩니다.
- 이 저장소에는 도구(`index.html`)만 포함됩니다. **인사 데이터(엑셀, 백업 JSON)는 절대 저장소에 커밋하지 마세요.** `.gitignore`에 기본 차단 규칙이 들어 있습니다.
- 자동 저장은 같은 PC·같은 브라우저에서만 유지됩니다. 중요한 시점마다 백업 JSON을 내려받아 보관하세요.

## 기술 구성

- 단일 HTML 파일 (프레임워크 없음)
- [SheetJS](https://cdnjs.com/libraries/xlsx) — 엑셀 읽기 (날짜는 일련번호로 파싱하여 시간대 영향 제거)
- [ExcelJS](https://cdnjs.com/libraries/exceljs) — 서식 포함 엑셀 생성 (UTC 날짜 기록으로 시간대 밀림 방지)

휴가대장 만들기

파일 2개를 올리고 연도를 선택하면 버튼이 활성화됩니다.
계산 방식과 시트 서식(색상·테두리·병합·표시형식)은 기존 휴가대장(2026) 시트와 동일합니다.
생성된 시트의 숫자는 계산 결과가 고정된 값입니다. 데이터가 바뀌면 다시 생성해 주세요.

사용 설명서

처음 시작하기

① 인원DB 양식 받기 / 휴가DB 양식 받기로 엑셀 양식을 내려받아 데이터를 채웁니다.
② 1단계에 두 파일을 각각 올립니다.
③ 연도를 고르고 엑셀 파일 생성을 누르면 끝!

기존에 쓰던 통합 엑셀(인원DB·휴가DB 시트 포함)이 있다면 그 파일 하나를 양쪽에 똑같이 올려도 됩니다. 필요한 시트를 자동으로 찾아냅니다.

자동 저장

파일을 올리거나 데이터를 수정하면 이 브라우저에 자동 저장됩니다. 다음에 이 파일을 다시 열면 마지막 데이터가 그대로 복원되어, 파일을 매번 다시 올릴 필요가 없습니다.

  • 같은 PC · 같은 브라우저에서만 유지됩니다.
  • 화면 위 상태바의 초록 점이 켜져 있으면 저장이 동작 중입니다.
시크릿 창에서 열거나, 브라우저의 "인터넷 사용 기록 삭제"로 사이트 데이터를 지우면 저장된 내용이 사라집니다. 중요한 시점마다 백업 내려받기를 해 두세요.

백업과 복원

백업 내려받기 — 현재 인원DB·휴가DB 전체를 JSON 파일 하나로 저장합니다. 다른 PC로 옮기거나 보관용으로 쓰세요.
백업 복원 — 백업 JSON 파일을 선택하면 그 시점의 데이터로 되돌립니다. (현재 데이터는 덮어써집니다)
저장 데이터 초기화 — 브라우저에 저장된 데이터를 지우고 처음 상태로 돌아갑니다.

데이터 수정 (2단계)

  • 셀을 클릭해 바로 수정합니다. 수정 즉시 자동 저장됩니다.
  • ＋ 추가 버튼으로 인원/휴가기록을 추가, 행 끝 ✕로 삭제합니다.
  • 휴가DB 탭의 검색창에서 성명·연도·휴가구분으로 찾을 수 있습니다.
  • 수정한 인원DB/휴가DB 내려받기로 엑셀 파일로도 보관할 수 있습니다.

날짜 · 입력 형식

  • 날짜는 2026-06-12 형식 (2026.06.12, 2026/06/12도 인식)
  • 퇴사일이 없으면 - 또는 빈칸
  • 사용일은 숫자 — 하루 1, 반차 0.5
  • 휴가DB의 YEAR를 비워두면 신청일의 연도가 자동 적용됩니다.

집계 규칙 (기존 수식과 동일)

휴가구분집계
연차 · 반차 · 여름휴가연차에서 차감
월차월차에서 차감 (2017.5.30 이후 입사자, 입사 12개월간 최대 11일)
병가 · 예비군 · 민방위 · 경조휴가 · 특별휴가별도 항목으로 집계
대체휴무(반차/연차) · 무급휴가 · 만료된 연차연도별 내역에만 표시
그 외 모든 구분기타휴가로 집계 + 기타휴가(연도) 시트에 목록 표시

연차 발생: 근속 1년당 15일, 2년마다 1일 가산, 최대 25일.
정산기준일: 지난 연도는 그해 12월 31일, 올해는 오늘.

노란 경고 상자의 의미

  • 기타휴가로 집계되는 휴가구분 — 표준 구분명과 다르게 입력된 항목입니다. 의도한 것이면 그대로 두고, 아니면(예: 경조사 → 경조휴가) 2단계에서 수정하세요.
  • 인원DB에 없는 성명 — 휴가기록은 있는데 인원DB에 없는 사람입니다. 대장에 나타나지 않으니 인원DB에 추가하거나 이름 오타를 확인하세요.

자주 묻는 질문

Q. 저장이 안 된다고 떠요.
이 파일을 PC에 내려받아 더블클릭으로 열어 주세요. 일부 미리보기 환경이나 보안 설정에서는 브라우저 저장이 막혀 있습니다. 그 경우에도 백업 내려받기/복원으로 데이터를 이어갈 수 있습니다.

Q. 다른 PC에서 쓰려면?
이 HTML 파일과 백업 JSON을 함께 옮긴 뒤, 새 PC에서 백업 복원을 누르세요.

Q. 생성된 엑셀 숫자가 안 바뀌어요.
생성 파일은 계산 결과가 고정된 값입니다. 데이터 수정 후 다시 생성해 주세요.
